Pacific Trails Resort

Throughout this course you will need to complete the Pacific Trails Resort case study at the end of each chapter. These case studies serve as a way to reinforce the skills discussed within each chapter. At the end of each week, you will be required to create a ZIP file of your case study folder and submit it via the submission link below. Each chapter's case study (FYI - Not all chapters have an end of chapter case study) will build upon the previous chapter's case study files. You will need to submit the LAST chapter of the week's case study files when you submit or you will not get full credit.

Solutions videos to the prior week's case studies will be posted the following week. Please review the solution video and make sure you correct your files as needed.

Textbook error:

  • Figures 6.33, 6.34, and 6.35 should have no white space on the left or the right side of those hero images.  

Tips:

  1. Practice Hands-On Practice 6.6 (pages 198-199) and Practice 6.10 (page 207). Solution files can be downloaded from https://www.pearson.com/us/higher-education/product/Felke-Morris-Student-Files-for-Basics-of-Web-Design-HTML-CSS-5th-Edition/9780135228364.html

  3. When you include image file name in the background-image property in CSS file, no img tag in HTML files is needed. Otherwise, the image will be displayed twice.
  4. Make sure you include the main element selector in pacific.css in all HTML files.
  5. Please watch the case study solution video(s) to make sure files from your previous weeks are correct before you continue to do this week's case study assignment.
