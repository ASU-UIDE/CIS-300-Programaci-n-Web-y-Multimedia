Yes, a YouTube video can be embedded in the project assignment.  However, the project assignment is required to include a video tag in one of the html files.  So a youtube video can be embedded but will NOT be counted as the video points. 

You can find any video online or create your own video to be used with the video tag.  If you will create your own video, make sure it is less than 1 minute so we won't run into any problems in the final project submission.  The larger file does take longer time to be submitted.  The video should be in the mp4 format or any other formats that Chrome supports.  A poster image is highly recommended.  

In addition to the video, included in the video tag, you can embed a YouTube video in the project assignment with the iframe tag.  An iframe is used to display a web page within a web page.  More info about iframe can be found at https://www.w3schools.com/html/html_iframe.asp

Following the steps in https://support.google.com/youtube/answer/171780?hl=en  to embed YouTube video to your project website.
