Common mistakes:

  • My nav didn't work:
    • The file name in the assignment folder needs to be the same as the file name in the a href field.
    • For example, the name in the a tag href field is "index.html", then in yourasurite-ch#CS folder, the file name needs to be index.html. Index.html will not work. It is case-sensitive.The double quote in the href field should be two short straight vertical bar ".
    • If you see any curved double quote, delete and retype it to make sure it is ". Curved double quotes can't be recognized by the browser.
  • I submitted the wrong files:
    • After submitting your assignment file to Canvas, please go to My Grade page to download the file your have submitted to make sure you have submitted the correct file.

      2. Move the downloaded file to desktop
