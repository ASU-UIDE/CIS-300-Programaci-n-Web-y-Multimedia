The discussion boards are a hugely valuable tool for everyone in that it can allow:

  • Others to benefit from the answers we receive to our questions (it's rare for anyone to be the only person with a question).
  • Someone with a problem to get help faster (I try to answer questions as soon as I can, but I can't answer as fast as everyone can).
  • Students to learn by helping others (a common saying with teachers is that you don't truly understand something until you can explain it to others).

So I encourage you to try the discussion boards first with your questions and let's see if we can create a supporting community for the course.

Best practices when posting:

  • Try hard not to ask the same question again. Read over the posts broadly before asking a new question. By not posting the same question multiple times, the discussion board will be easy to search and find answers.
  • Be as specific and detailed as possible. It is really hard to help someone who "Can't upload the assignment." It is much easier to help someone who "Gets an error message that says X when trying to submit assignment Y while using Firefox on my Windows 10 PC after doing the assignment in Cloud9."
  • Don't pile on - Multiple replies and posts of "Me too" are generally not productive.
  • Strive to be extra courteous. It is very easy for a comment to be interpreted poorly. Remember, you are all hear to learn new skills, be respectful when replying.
