Wireframe Project Assignment

Q:  Do we need to create a wireframe for each page on the site or just the home page? 

A: Just the Home page or one of the other pages, but make sure you include as much as info you can based on the instruction in the project doc. The purpose of the wireframe is to display your website on paper. It is easier to change the wireframe vs. to change the HTML and CSS code. If you are working for a client, you can put a wireframe together so you can communicate with your client before you start creating the code. 

Q:  How detail the information should we include?

A:  We need to have as detail info as possible.  For example, do NOT just write a "header" in the header section but specify what kind info will be included in the header section, e.g. logo, company name, company picture, a slogan, etc.  If you will not have a logo in the website, make a note for that. 

In the nav section, list all the nav items which should be matched to the sitemap blocks. 

In the main section, display where the hero image, if there is, text, or video/form/table are located. 

In the footer section, do NOT just write a "footer" in that section but specify what info will be included in that section, e.g.  email, copyright, social media icons, company contact info, or some nav items, etc. 

Review the attached rubric to see how the assignment will be graded.

Q: How can I test the responsive Layout?

A: Chrome developer tool can be used to test the responsive Layout and troubleshoot other problems we may have.  To open the developer tool, first open the html file in a Chrome browser, right click your mouse, then click Inspect.  To view the html file in a Responsive layout, click the "Toggle Device Toolbar" icon   It is located on the right side of the Elements item above the html code.  In the option bar, above the page view, we can change the width (e.g. from 1024 to 600) as well as 100% to 50% (for better viewing the whole page).  We can also just drag the two vertical bars icon on the right side of the page view to make it wider or narrower.
