To use Chrome Developer Tools

If you are using a text editor, view your html file in Google Chrome browser. Right click on the html file and open it in Google Chrome.

Once your html file is displayed in a chrome browser, move your mouse to the page location where you are having "troubles, right click to select "Inspect" to open Google Chrome Developer Tools. You will see both the html code under the Elements option and CSS code under the Styles option. It is a great tool to see which style properties have been applied to those html code. Using Chrome Developer Tools: Elements (1/6) provides more info in how to use Google Chrome Developer Tools.
