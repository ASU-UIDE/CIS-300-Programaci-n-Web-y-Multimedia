Following the following steps to verify the assignment zipped file:

  1. Unzip the file
  2. Move the yourasurite-ch#CS folder to your desktop (e.g. selin2-ch2CS folder)
  3. Open the yourasurite-ch#CS folder to find a HTML file (e.g. index.html)
  4. Open that HTML file from a text editor then run it in a Chrome browser or open it with Google Chrome directly

Please NOTE:  hyperlinks in the nav section will NOT work if those html files are in the zipped file.
