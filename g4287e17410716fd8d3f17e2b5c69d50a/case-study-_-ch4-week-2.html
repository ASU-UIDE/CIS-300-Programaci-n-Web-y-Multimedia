Pacific Trails Resort

Throughout this course you will need to complete the Pacific Trails Resort case study at the end of each chapter. These case studies serve as a way to reinforce the skills discussed within each chapter. At the end of each week, you will be required to create a ZIP file of your case study folder and submit it via the submission link below. Each chapter's case study (FYI - Not all chapters have an end of chapter case study) will build upon the previous chapter's case study files. You will need to submit the LAST chapter of the week's case study files when you submit or you will not get full credit.

Solutions videos to the prior week's case studies will be posted the following week. Please review the solution video and make sure you correct your files as needed.

Tips:

  • Practice Hands-On Practice 4.6 (pages 128-129).Solution files can be downloaded from https://www.pearson.com/us/higher-education/product/Felke-Morris-Student-Files-for-Basics-of-Web-Design-HTML-CSS-5th-Edition/9780135228364.html
  • The closing html tag is the last line of code in all HTML files.
  • The closing body tag is the 2nd last line of code in all HTML files.
  • Include a Link element, in the head section, in all HTML files with pacific.css as the value in href attribute.
  • For the external style sheet file pacific.css, you don't need style tag or closing style tag in it. Style tag is only needed with embedded CSS code in html files. See page 120-123 for more information.
  • Please watch the case study solution video to make sure files from your previous week are correct before you continue to do this week's case study assignment.
